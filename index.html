About.

We made REMify because this is 2023. Web Development has came a long way and its time too leave pixels in the past. Lets learn the new modern CSS sizing methods. REM, EM, VH, etc.

How REMify Works.

A user enters a REM/Pixels amount and our program determines which method too use, then it does some calculations and returns the specified amount too the user.
For example lets say we (a user) enters 500 Pixels and we want too change this too rem.
Our program would take that amount and divide it by 16 and return: 31.25 REM.
